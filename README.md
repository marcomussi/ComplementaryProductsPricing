# Online Dynamic Pricing of Complementary Products

#### *Marco Mussi and Marcello Restelli*

This repository contains the code used to run the experiments presented in the paper "Online Dynamic Pricing of Complementary Products".

#### Required Packages

Run this code requires *numpy*, *matplotlib*, *ortools*, *tqdm*, *tikzplotlib*
